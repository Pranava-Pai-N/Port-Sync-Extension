# Change Log

All notable changes to the PortSync extension are documented in this file.

See [Keep a Changelog](http://keepachangelog.com/) for recommendations on how to structure this file.

## [Released]

## [0.0.1] - 03-06-2026

- Initial product release delivering a VS Code experience for tracking active development ports in real time
- Added a dedicated local sidebar view with broad compatibility across Windows, Linux, and macOS
- Included comprehensive README documentation with installation, usage, development, and contribution guidance
- Added developer information, clear license details, and extension command metadata to improve discoverability and onboarding


## [0.0.2] - 07-06-2026
- Added search functionality for quick retrieval of active ports from large number of ports.
- Minor bugs fixed and new features added as per the requirements.
- Minor UI Fixes implemented along with the inital release fixes.
